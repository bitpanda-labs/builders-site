Get Started

Bitpanda CLI

A fast, scriptable command-line tool for the Bitpanda Developer API. View your portfolio, check live prices, and browse trades and transactions — all from your terminal.

Read-only. All commands are read-only. The CLI cannot place orders, move funds, or modify your account.

Install

Pick the install method that suits your platform.

shell
brew install bitpanda-labs/tap/bp

Available for macOS and Linux via Homebrew.

shell
sudo dpkg -i bp_*_linux_amd64.deb
shell
sudo rpm -i bp_*_linux_amd64.rpm

Configure

bp needs a Bitpanda API key. Generate one at web.bitpanda.com/apikey.

Provide the key in one of three ways (highest priority first):

  1. Per-command flag

    Pass --api-key YOUR_KEY to any command. Useful for one-off scripts.

  2. Environment variable

    Export BITPANDA_API_KEY in your shell. Best for CI and quick interactive use.

  3. Config file

    Save the key to ~/.config/bitpanda/config.yaml with permissions 0600. Best for daily desktop use — set once and forget.

config file setup
mkdir -p ~/.config/bitpanda
echo "api_key: YOUR_KEY" > ~/.config/bitpanda/config.yaml
chmod 600 ~/.config/bitpanda/config.yaml
Keep your key safe. The config file holds your API key in plaintext — restrict it to 0600. bp warns on stderr if permissions are looser.

Commands

Every command supports -o table (default), -o json, and -o csv. Run bp --help or bp <command> --help for full flag reference.

portfolio
bp portfolio

Aggregated holdings with EUR valuations, sortable.

balances
bp balances

Raw wallet list with filtering and pagination.

trades
bp trades

Buy/sell history with type, asset, and date filters.

tx
bp transactions

Full transaction log including rewards and staking.

price
bp price <SYMBOL>

Live price and daily change for a single asset.

prices
bp prices

Prices for held assets, or all available with --all.

asset
bp asset <UUID>

Resolve an asset ID to its name and symbol.

completion
bp completion

Generate shell completions (bash, zsh, fish, PowerShell).

Examples

shell
bp portfolio --sort value          # biggest positions first
bp price BTC                       # single asset
bp prices --all                    # every available price
bp trades --operation buy --from 2024-01-01
bp transactions --flow incoming -o csv > tx.csv
bp portfolio -o json | jq '.[] | select(.eur_value > 100)'

Output formats

FormatFlagUse case
table-o table (default)Human-readable terminal output
json-o jsonPipe to jq, scripts, automation
csv-o csvSpreadsheets and tax tools

Exit codes

CodeMeaning
0Success
1General error
2Authentication error (bad or missing API key)
3API error (Bitpanda returned non-2xx)

Shell completion

Add bp auto-completion to your shell. Pick the matching shell:

shell
source <(bp completion bash)
shell
source <(bp completion zsh)
shell
bp completion fish | source
powershell
bp completion powershell | Out-String | Invoke-Expression

To make it permanent, append the matching command to your shell rc file (~/.bashrc, ~/.zshrc, etc.).
