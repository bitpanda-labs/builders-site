Bitpanda Fusion API

Bitpanda's advanced trading interface for developers and automated traders. Place orders, manage positions, stream live orderbook data, and access full trade history via REST and WebSocket.

Full documentation is hosted at docs.fusion.bitpanda.com.

Generate an API key at web.bitpanda.com/apikey to get started. Pass it as Authorization: Bearer YOUR_KEY on every request.
