Get Started

Bitpanda MCP Server

Connect your AI assistant — Claude, ChatGPT, Cursor, or any MCP-compatible client — directly to your Bitpanda account. Ask about your portfolio, check live prices, and review trade history in plain language.

Read-only by design. All tools are strictly read-only. The server cannot place orders, move funds, or modify your account in any way.

How it works

The Model Context Protocol (MCP) is an open standard that lets AI assistants securely connect to external services. The Bitpanda MCP server bridges your AI tool and the Bitpanda Developer API, giving your assistant read-only access to your account data.

For consumer AI clients (Claude, ChatGPT, AI agents), authentication is handled via OAuth — you authorize the connection through your Bitpanda account with no manual key management. For technical setups and self-hosted deployments, direct API key authentication via the X-Api-Key header is also supported.

Quick start

Most users connect via OAuth in under a minute. Pick your client below.

  1. Open the connector directory

    In ChatGPT, go to Settings → Connectors (or click the + menu in a chat → Add apps & connectors).

  2. Find Bitpanda

    Search for Bitpanda in the directory and click Connect.

  3. Authorize

    ChatGPT redirects you to Bitpanda to log in and approve access. No API key required.

  1. Open Connectors

    In Claude Desktop, go to Customize → Connectors.

  2. Find Bitpanda

    Browse the directory and click Connect on the Bitpanda entry.

  3. Authorize

    Claude opens a browser. Log in to Bitpanda and approve access — no API key required.

TERMINAL 
claude mcp add --transport http bitpanda https://mcp.bitpanda.com/mcp
Claude will open a browser to authorize your Bitpanda account via OAuth. No API key required.

ChatGPT

The Bitpanda MCP server is published in the ChatGPT connector directory. Install it directly from ChatGPT.

  1. Open the connector directory

    In ChatGPT, go to Settings → Connectors, or click the + menu in any chat and choose Add apps & connectors.

  2. Search for Bitpanda

    Find Bitpanda in the directory and click Connect.

  3. Authorize via Bitpanda

    ChatGPT redirects you to Bitpanda's login page. Sign in and approve the connection — Bitpanda issues a short-lived token and ChatGPT manages it for you.

  4. Start asking

    Try: "What is my current BTC value?" or "Summarize my trades from last month." ChatGPT will prompt you to allow the Bitpanda connector to be used.

Connectors are available on ChatGPT Plus, Pro, Team, Enterprise, and Edu plans. To revoke access at any time, remove the connector from Settings → Connectors.

Claude Desktop

The Bitpanda MCP server is listed in Claude's connector directory. Add it from inside the app — no JSON config or restart required.

  1. Open Connectors

    In Claude Desktop, go to Customize → Connectors. On Team or Enterprise plans, an organization owner first needs to add the connector under Organization settings → Connectors.

  2. Find Bitpanda

    Browse the directory and click Connect on the Bitpanda entry.

  3. Authorize via Bitpanda

    Claude opens a browser tab. Sign in to Bitpanda and approve the connection — Claude stores and refreshes the token automatically.

  4. Enable it in a chat

    Open a conversation, click the + button at the lower-left of the chat, and choose Connectors to enable Bitpanda for that chat. Try: "Show me my top holdings."

Connectors are available on Claude Pro, Max, Team, and Enterprise plans. Manage or remove the connector at any time from Customize → Connectors.

Claude Code

  1. Add the server

    Run the command below in your terminal.

Cursor / VS Code / Windsurf

Available tools

All tools are read-only and annotated with readOnlyHint: true.

portfolio
get_portfolio

Aggregated portfolio with EUR valuations. Optional sort_by: value (default) or name.

market-data
get_price

Current EUR price for a single asset by ticker symbol — BTC, ETH, SOL, and more.

market-data
list_prices

Ticker prices for all held assets. Set all_assets=true for the full market-wide list. Supports limit.

market-data
get_asset

Asset metadata (name, symbol, type) by asset UUID.

wallets
list_wallets

Asset wallet balances. Filter by non_zero, asset_id. Supports limit and page_size.

transactions
list_transactions

Unified transaction history — deposits, withdrawals, transfers, trades. Filters: wallet_id, flow, asset_id, from_including, to_excluding. Set all=true to fetch all pages.

trades
list_trades

Buy/sell trade history derived from transactions. Filters: trade_type (buy/sell), asset_type, date range. Set all=true to fetch all pages.

API key setup Advanced

Direct API key authentication is available for custom scripts, programmatic integrations, and clients that do not support OAuth. Pass your Bitpanda API key in the X-Api-Key header with each request.

If your client supports OAuth, prefer that — it requires no key management and limits exposure. Use API keys only when OAuth is not available.

Generate your API key

  1. Log in to Bitpanda

    Go to web.bitpanda.com and sign in.

  2. Open API key settings

    Navigate to web.bitpanda.com/apikey or Profile → API key.

  3. Generate a new key

    Click Generate new API key. Copy it — you won't see it again.

Use with any MCP client

MCP CLIENT CONFIG 
{
  "mcpServers": {
    "bitpanda": {
      "url": "https://mcp.bitpanda.com/mcp",
      "headers": {
        "X-Api-Key": "YOUR_BITPANDA_API_KEY"
      }
    }
  }
}

FAQ

Do I need a Bitpanda account?

Yes. You need an active Bitpanda account. For OAuth-based clients (Claude, ChatGPT) just log in when prompted. For API key access, generate a key at web.bitpanda.com/apikey.

Can my AI assistant place trades or move funds?

No. All tools are strictly read-only (readOnlyHint: true). The server cannot place orders, withdraw funds, or modify your account in any way.

How does OAuth authentication work?

When you connect via Claude, ChatGPT, or another OAuth-enabled client, the client redirects you to Bitpanda's login page. After you log in and approve access, Bitpanda issues a short-lived token to the MCP server — your Bitpanda password is never shared with your AI client.

The token is stored securely by your AI client and refreshed automatically.

What is MCP?

The Model Context Protocol is an open standard for connecting AI assistants to external data sources. Supported by Claude, ChatGPT, Cursor, VS Code Copilot, Windsurf, and more.

Is this the official Bitpanda MCP server?

Yes. Maintained by Bitpanda Labs — the official MCP integration for the Bitpanda platform.

Do I need to install anything for OAuth-based clients?

No local installation required. Add https://mcp.bitpanda.com/mcp to your client config and authorize via browser. The server runs on Bitpanda's infrastructure.
