Get Started

Bitpanda MCP Server

Connect your AI assistant — Claude, ChatGPT, Cursor, or any MCP-compatible client — directly to your Bitpanda account. Ask about your portfolio, check live prices, and review trade history in plain language.

How it works

The Model Context Protocol (MCP) is an open standard that lets AI assistants securely connect to external services. The Bitpanda MCP server bridges your AI tool and the Bitpanda Developer API, giving your assistant access to your account data.

Authentication is via your Bitpanda API key — generate one and pass it with your client config. OAuth support (no key required) is coming soon for Claude Desktop, ChatGPT, and other connector-directory clients.

API key setup

Generate a Bitpanda API key and add it to your client config. Takes under a minute.

  1. Log in to Bitpanda

    Go to web.bitpanda.com and sign in.

  2. Open API key settings

    Navigate to web.bitpanda.com/apikey or Profile → API key.

  3. Generate a new key

    Click Generate new API key. Copy it — you won't see it again.

Claude Code

Claude Code is Anthropic's CLI for AI-assisted development. Add the MCP server with a single command.

  1. Add the server

    Run the command below in your terminal, substituting your API key.

  2. Verify

    Run claude mcp list to confirm the server is registered.

  3. Start asking

    Try: "What is my current BTC value?" or "Summarize my trades from last month."

TERMINAL 
claude mcp add --transport http bitpanda https://mcp.bitpanda.com/mcp \
  --header "X-Api-Key: YOUR_BITPANDA_API_KEY"
VERIFY 
claude mcp list

Cursor / VS Code / Windsurf

Most modern AI editors support remote MCP servers. Add the server URL and your API key to the config file.

Cursor (.cursor/mcp.json)

.cursor/mcp.json 
{
  "mcpServers": {
    "bitpanda": {
      "url": "https://mcp.bitpanda.com/mcp",
      "transport": "http",
      "headers": {
        "X-Api-Key": "YOUR_BITPANDA_API_KEY"
      }
    }
  }
}

VS Code (.vscode/mcp.json)

.vscode/mcp.json 
{
  "servers": {
    "bitpanda": {
      "type": "http",
      "url": "https://mcp.bitpanda.com/mcp",
      "headers": {
        "X-Api-Key": "YOUR_BITPANDA_API_KEY"
      }
    }
  }
}

FAQ

Do I need a Bitpanda account?

Yes. You need an active Bitpanda account. Generate an API key at web.bitpanda.com/apikey.

What is MCP?

The Model Context Protocol is an open standard for connecting AI assistants to external data sources. Supported by Claude, ChatGPT, Cursor, VS Code Copilot, Windsurf, and more.

Is this the official Bitpanda MCP server?

Yes. Maintained by Bitpanda Labs — the official MCP integration for the Bitpanda platform.

Is my API key sent to the AI model?

No. Your API key lives in your client config and is used server-side to authenticate requests to Bitpanda. The model never sees it.
